>   - after d4 -> exd4 
>       - 3.exd4 4. Bxf7+ Kxf7 5. Qh5+ g6 6. Qxc5

> line: Bxd4 4. Nf3 Nc6 5. O-O Nf6 6. Nxd4 Nxd4

>    - after Nxd4 -> exd4
>       - 7. e5 Ne4 8. Qg4
